## Overview
This repository contains my Digital Literacy Project for CSE0001 at VIT Bhopal. The project is structured as a portfolio demonstrating essential digital skills required for students, including awareness, professional online presence, platform usage, communication, and cybersecurity.

It is organized into five module-based tasks:
1. Digital literacy infographic.
2. Student digital portfolio setup.  
3. Exploration of coding and collaboration platforms.  
4. Professional email writing and social media etiquette.  
5. Cybercrime case study with prevention strategies.

Each task includes relevant artifacts such as screenshots, documents, and reflections, along with a consolidated project report.
